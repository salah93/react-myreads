# MyReads
My Bookshelf ~*~*~

## Requirements
+ npm
+ git
+ python
+ flask
+ pymongo
+ mongodb-server

## Steps to launch app
+ clone repo
`git clone https://github.com/salah93/myreads.git`
+ go into directory
`cd myreads`
+ install dependencies
`npm install`
+ start app
```
 rm -rf static templates
 npm run build
 python app.py
```
